# Leftovers Recipe Generator 🍳

Turn your fridge leftovers into delicious meals! Select ingredients you have, and get personalized recipe suggestions.

## Features

- ✅ **Ingredient Checklist** - Click to select what's in your fridge
- 🍲 **Smart Recipe Suggestions** - Get 1-2 recipes using your ingredients
- 📱 **No Account Needed** - Use instantly without registration
- 💾 **Save Favorites** - Create an account to save your favorite recipes
- 🔗 **Shareable Links** - Share your ingredient combos with friends
- 🎨 **Elegant UI** - Beautiful, classy design inspired by premium templates

## Tech Stack

**Frontend:** React.js, React Router, Axios, React Share  
**Backend:** Node.js, Express.js, MongoDB, JWT Authentication  
**Styling:** CSS3 with custom components

## Installation

### Prerequisites
- Node.js (v14 or higher)
- MongoDB installed locally or MongoDB Atlas account
- npm or yarn

### Backend Setup

```bash
# Navigate to backend folder
cd backend

# Install dependencies
npm install

# Create .env file with your configuration
# PORT=5000
# MONGODB_URI=mongodb://localhost:27017/leftovers_recipe
# JWT_SECRET=your_secret_key_here

# Start the backend server
npm run dev
